I'm a final-year **Biomedical Engineering** student at **İzmir Katip Çelebi University**, turning my interest in artificial intelligence, software development, and embedded systems into engineering solutions through **TÜBİTAK** and **TEKNOFEST** projects.

- 🔬 Undergraduate researcher at the **Biomedical AI & Neuroscience Lab (İKÇÜ)** — advisor: Dr. Mehmet Akif Özdemir
- 🌍 Building **MEDAIGENCY / KillianAI** (Interreg EU): LLM-based autonomous agents for disaster response
- 🧠 Developing **Genomic-AI** for **TEKNOFEST 2026 Health AI** — hybrid Transformer–XGBoost stacking ensemble for missense variant pathogenicity (**F1 = 0.898 · ROC-AUC = 0.951**)
- 🩻 Working on **FossaNET** — real-time NIR forearm vein segmentation (MAnet + EfficientNet-B1, **Dice = 0.737**)
- 🎓 Biomedical Engineering Dept. Representative · Faculty Quality Commission Representative

## 🚀 Featured Projects

### 🧬 Genomic-AI — TEKNOFEST 2026 Health AI Competition
> Hybrid **Transformer–XGBoost stacking ensemble** for missense variant pathogenicity classification.  
> **Team Duo-DX** · Advisor: Dr. Mehmet Akif Özdemir  
> 🎯 **F1 Macro: 0.898 · ROC-AUC: 0.951**  
> Working on the full multi-panel competition dataset (Master, Kanser, PAH, CFTR) with panel-specific threshold optimization and adversarial validation.

### 🩻 FossaNET — Real-time NIR Vein Segmentation *(Thesis)*
> Lightweight semantic segmentation on the **CUBITAL** dataset (2,016 images).  
> 🎯 **MAnet + EfficientNet-B1 · Dice = 0.7374 · 6.5 FPS on CPU**  
> Two-track strategy: nnU-Net for academic results + MobileNetV2-UNet TFLite INT8 for Raspberry Pi 5 deployment.

### 🌍 MEDAIGENCY / KillianAI — Interreg EU Project
> Autonomous tactical GIS assistant for disaster response.  
> LLM-based multi-agent system · PostGIS · OSRM routing · MapLibre GL JS · TOPSIS-based source selection.

### 💉 Infrared Vein Imaging Device — TÜSEB A1 / TÜBİTAK 2209
> Image processing algorithms and full technical design for a clinical vein-visualization device.  
> 🔗 [github.com/berkantaksyy/vein-visualization](https://github.com/berkantaksyy/vein-visualization)

### 🧪 GlucoseDetect AI
> Random Forest model estimating glucose concentration from non-enzymatic sensor data.  
> 🎯 **98% accuracy** · Flask web interface · end-to-end analysis pipeline.

### 📡 Real-Time Wireless ECG Application
> Android (Kotlin) app receiving high-frequency ECG from Raspberry Pi over Wi-Fi (TCP/IP socket).  
> On-device peak detection · live BPM · CSV logging · local PDF reporting.

### 🩹 EMG Signal Processing & Analysis Software
> Python desktop app for raw EMG signals: dynamic digital filtering, RMS, and FFT spectral analysis.

### 🛩️ Papalagi UAV — TEKNOFEST
> Autonomous flight algorithms (Python / Linux) for full mission-scenario completion.
